# 銀河創業大冒險 — 台灣公司行號設立流程互動遊戲

## 檔案說明
- `taiwan-company-game.html` — 完整單一 HTML 遊戲檔案（self-contained，不需後端）

---

## 遊戲結構（共 7 關）

### 角色選擇頁（登入系統）
- 4 個角色：通訊指揮官、艦隊艦長、星際商人、太空探索家
- 角色圖像直接嵌入（base64），來自用戶提供圖片
- 輸入暱稱後才可啟航

### 第 1 關：公司 or 行號？
- 3 題情境單選題，根據答案推薦「有限公司」或「行號」
- 顯示對應價格：公司辦到好 NT$6,000 / 行號辦到好 NT$4,000
- 提供「重新作答」選項

### 第 2 關：挑選營業項目
- 12 個預設項目（勾選題，可複選）
- 自由輸入欄位（可自訂項目）
- 特許行業提醒（醫療器材、室內裝修、寵物業、菸酒販售）

### 第 3 關：名稱預查
- 公司路徑：全國不能重複，需有組織型態結尾
- 行號路徑：同縣市不能重複，需有業態字尾
- 各一題選擇題，答錯有解析

### 第 4 關：準備文件 + SVG 連連看
- 公司：存入資本額 → 銀行餘額證明 → 會計師驗資報告 → 屋主同意書
- 行號：存入資本額（超過25萬才需）→ 屋主同意書
- 連連看：點左邊 → 點右邊，SVG 貝茲曲線從右邊緣連到左邊緣（不畫到文字）

### 第 5 關：翻牌歸類小遊戲
- 6 張卡牌，各標示公司/行號特徵
- 點選「公司」或「行號」歸類，答對顯示解析
- 卡牌內容：審查1-2週、審查3-5天、需驗資報告、不需驗資報告、可申請免用統一發票、資本超過25萬才需存款

### 第 6 關：稅籍登記
- 說明負責人需親赴國稅局，填寫開業日期
- 免開發票門檻選擇題：答案為未達 20 萬元

### 第 7 關：開張前 Checklist
- 4 項：銀行帳戶、記帳委託、電子發票/免開發票、勞健保
- 全部打勾後顯示獎狀 + 截圖提示 + LINE 聯絡按鈕

## 視覺風格
- 宇宙銀河深空主題（深藍/黑底 + 星點動畫 + 星雲）
- 字體：Orbitron（標題）+ Exo 2（內文）
- 主色：#40B4FF（藍）、#C8A8FF（紫）、#7FFFDF（青）
- 公司色：#6C5CE7（紫）、行號色：#34D399（綠）

---

## 重要設定項目

### LINE 連結
在 HTML 中搜尋：
```
https://lin.ee/QMmohTiz
```
修改此 URL 即可更新 LINE 聯絡按鈕。

### 服務價格
在 HTML 中搜尋：
```
行號辦到好 NT$4,000
公司辦到好 NT$6,000
```

### 免開發票門檻
第 6 關答案為 `C`（未達 20 萬元），在 `s6Check` 函式中 `const correct=2`。

---

## 技術細節
- **純前端**：單一 HTML 檔案，無需伺服器
- **角色圖片**：base64 嵌入，來自 IMG_8248.png 裁切 + 背景去除
- **進度系統**：7 關進度條 + 步驟點
- **每關均有「跳過此關」按鈕**（左側虛線樣式），下一步按鈕在右
- **手機友善**：RWD 設計，按鈕尺寸手機可點

## 開發歷程摘要（Claude 對話紀錄）

1. 建立基礎 7 關遊戲架構（HTML/CSS/JS 單檔）
2. 加入宇宙銀河視覺風格（Orbitron 字體、星空動畫、深色卡片）
3. 角色選擇頁（6 角色 → 縮減為 4 角色）
4. 角色圖片從用戶提供圖片裁切、背景去除（scipy flood-fill 算法）
5. 第 4 關連連看升級為 SVG 曲線連線（edge-to-edge，不蓋字）
6. 第 5 關改為翻牌歸類小遊戲
7. 多次迭代修正角色裁切（艦長、外星商人邊界精準定位）
8. 修正 charImg 函式遺失導致的第一頁錯誤
9. 各關文字內容校正（免開發票門檻 20 萬、屋主同意書兩者皆需等）
